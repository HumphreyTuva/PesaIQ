# Project Roadmap

This document outlines the planned features and improvements for PesaIQ. The roadmap is divided into short-term, medium-term, and long-term goals.

## 🟢 Phase 1: Core Enhancements (Short-term)
- **Dark Mode Support:** Implementation of full Material 3 dynamic color and dark theme support for better night-time usability.
- **Enhanced Categorization:** Improve the rule-based engine to handle more edge cases and allow user-defined custom keyword rules.
- **PDF Statement Import:** Allow users to upload official M-Pesa PDF statements to backfill historical data that might be missing from the SMS inbox.

## 🟡 Phase 2: Connectivity & Backup (Medium-term)
- **Cloud Backup:** Securely backup transaction data to a private cloud storage (e.g., Google Drive) to ensure data is preserved when switching devices.
- **Google Sheets Integration:** Export transactions directly to a specified Google Sheet for advanced personal accounting.
- **Django Backend Sync:** Develop a companion web dashboard using Django to view spending habits on a larger screen.

## 🔴 Phase 3: Intelligence & Ecosystem (Long-term)
- **AI Spending Insights:** Integrate on-device machine learning (ML Kit) to predict future spending patterns and suggest budget adjustments.
- **Multi-device Synchronization:** Real-time sync between multiple Android devices using a secure backend.
- **Bank SMS Support:** Expand the parsing engine to support notifications from major Kenyan banks beyond M-Pesa.

---
*Note: This roadmap is subject to change based on user feedback and technical feasibility.*
